# Reestar
A simple restaurant website built using HTML and CSS featuring food items, menu cards, and a contact form with a clean layout.


Live Demo: https://shyxmmmmmm.github.io/Reestar/


📌 Project: Reestar Restaurant Website

🧑‍💻 This project is a simple restaurant website called Reestar built using HTML and CSS. It displays food items with images, descriptions, and an order button along with a contact form section.

✨ Features:

• 🍽 Restaurant style landing page

• 🧭 Simple navigation menu (Home, Recipe, Contact)

• 📷 Food cards with images and descriptions

• 🛒 Order Now button for each food item

• 📋 Contact form with Name, Age, and Email fields

• 🎨 Clean card layout for food items

• 🖱 Hover effect for navigation links

🛠 Technologies Used:

• HTML5 – webpage structure

• CSS3 – styling and layout

• Inline-block layout – arranging food cards

• Basic form elements – contact section

⚙️ How it works:

1️⃣ Users land on the restaurant homepage

2️⃣ Food items are displayed as cards with images

3️⃣ Each card shows food name, description, and order button

4️⃣ Users can scroll to the contact section

5️⃣ Contact form allows users to send their details

🎯 Purpose of the Project:

This project helps practice basic webpage structure, card layout design, navigation menus, and form creation using HTML and CSS.
